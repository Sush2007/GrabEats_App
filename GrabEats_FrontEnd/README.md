# 🚀 GrabEats – AI-Powered Smart Seating System

GrabEats is an AI-driven real-time occupancy and table management system designed for high-traffic dining environments like food courts, canteens, and restaurants. It eliminates the need for manual “table hunting” by converting live camera data into actionable seating insights.

---

## 🧠 Problem Statement

In crowded dining spaces, users often spend 10–15 minutes searching for seats, leading to:
- Frustration and poor user experience  
- Congestion and inefficiency  
- Revenue loss for vendors  

---

## 💡 Solution

GrabEats digitizes physical seating spaces using AI and cloud technologies to provide:
- Real-time seat availability  
- Smart dining decisions  
- Improved space utilization  

---

## ⚙️ Tech Stack (Google-Focused)

- **Frontend:** Flutter / React  
- **Backend:** Firebase (Realtime DB / Firestore, Auth, Hosting)  
- **Computer Vision:** YOLO (via Roboflow Inference API)  
- **Maps Integration:** Google Maps API  
- **Cloud Architecture:** Scalable API-based system  

---

## 🔍 Core Features

- 🎯 Real-time seat availability tracking  
- 🤖 AI-based occupancy detection (person + ROI logic)  
- 📍 Location-based restaurant discovery (Google Maps)  
- 📊 Business dashboard for seating analytics  
- ⚡ Cloud-based, low-hardware deployment  

---


---

## 🧪 How It Works

1. Camera captures dining area  
2. AI model detects people in real-time  
3. ROI logic determines seat occupancy  
4. Data is pushed to Firebase  
5. Users view live seat availability via app/web
